# ✈️ Airline Booking Data Analysis Dashboard (Excel)

## 📌 Project Overview

This project presents an **interactive Airline Booking Dashboard built entirely in Microsoft Excel**.
The goal is to analyze booking behavior, customer preferences, and operational trends to generate actionable business insights.

---

## 🎯 Objectives

* Analyze booking completion vs abandonment rates
* Identify the most effective **sales channels**
* Understand customer preferences (meals, baggage, seats)
* Discover trends across **flight days and time categories**
* Highlight top-performing routes and trip types

---

## 📊 Dashboard Features

The dashboard is fully interactive using **Slicers** and includes:

### 🔹 KPIs (Top Cards)

* Total Bookings
* Total Passengers
* Average Purchase Lead Time
* Average Length of Stay
* Average Flight Duration

### 🔹 Analysis Sections

* 📈 Booking Completion Rate (Completed vs Abandoned)
* 🌐 Sales Channel Performance (Internet vs Mobile)
* 🧳 Passenger Preferences:

  * Extra Baggage
  * Preferred Seat
  * In-Flight Meals
  * All Services
* 🛫 Trip Type Distribution (OneWay, RoundTrip, CircleTrip)
* 📅 Booking Trends by Flight Day
* ⏰ Passenger Distribution by Flight Time (Morning, Afternoon, Evening)
* 🗺 Top 10 Routes by Number of Bookings

---

## 💡 Key Insights

* Majority of bookings come from the **Internet channel**
* **RoundTrip** is the most preferred trip type
* **Morning flights** have the highest passenger volume
* High percentage of **abandoned bookings** indicates a need to improve conversion strategies
* Strong demand for **extra baggage and in-flight services**
